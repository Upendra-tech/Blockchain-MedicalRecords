## Setup
1. Download Metamask Extension from chrome
2. Set your wallet to Rinkeby test network
3. Get ETH from Rinkeby Faucet

## How to Run System
```
npm run start
```

## How to Run Tests
```
npm run test
```

## Whenever there is a change in Solidity code, use these commands
```
cd ethereum
node compile.js
node deploy.js
```
